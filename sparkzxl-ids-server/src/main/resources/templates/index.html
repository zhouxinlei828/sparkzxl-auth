操作步骤
  • 服务发现：http://localhost:8080/.well-known/openid-configuration
  • 加密公钥：http://localhost:8080/.well-known/jwks.json
  • 获取授权：http://localhost:8080/oauth/authorize
  • 确认授权：http://localhost:8080/oauth/confirm
  • 获取/刷新Token：http://localhost:8080/oauth/token
  • 收回Token：http://localhost:8080/oauth/revoke_token
  • 用户详情：http://localhost:8080/oauth/userinfo
  • check session：http://localhost:8080/oauth/check_session
  • 授权异常：http://localhost:8080/oauth/error
  • 登录：http://localhost:8080/oauth/login
  • 退出登录：http://localhost:8080/oauth/logout
  • 第一步：根据 Grant Types 点击下方 table 中的 ”生成授权链接“，点击后会在下方生成不同类型的链接
  • 第二步：在后面的四个授权模式中，选择合适的链接点击访问
  • 第三步：如果是选择的授权码模式或者隐式授权模式，会回调回当前页面，并且 URL 中附带相关参数（回调参数会在下方”回调参数“中显示）。如果是授权码模式，可以根据 code 换取 token
  • 第四步：获取完 access_token 后，可以在”通用方法“中使用 access_token 进行相关操作

API

应用参数

回调参数

通用方法
  • http://localhost:8080/oauth/userinfo?access_token=
  • http://localhost:8080/oauth/revoke_token?access_token=
  • http://localhost:8080/oauth/token?grant_type=refresh_token&
  • http://localhost:8080/oauth/logout
授权码模式
  • 获取授权：http://localhost:8080/oauth/authorize?client_id=?&response_type=code&scope=?&redirect_uri=?&state=?
  • 获取授权（跳过确认授权）：http://localhost:8080/oauth/authorize?autoapprove=true&client_id=?&response_type=code&scope=?&redirect_uri=?&state=?
  • 获取token：http://localhost:8080/oauth/token?client_id=?&client_secret=?&grant_type=authorization_code&redirect_uri=?&code=?
  • 获取授权(PKCE)：http://localhost:8080/oauth/pcke/authorize?client_id=?
  • 获取token(PKCE)：http://localhost:8080/oauth/pkce/token?client_id=?&code=?&state=?
隐式授权模式
  • 获取授权：http://localhost:8080/oauth/authorize?client_id=?&response_type=token&scope=?&redirect_uri=?&state=?
密码授权模式
  • 获取token：http://localhost:8080/oauth/token?grant_type=password&username=test&password=test&client_id=?&client_secret=?&scope=?
客户端授权模式
  • 获取token：http://localhost:8080/oauth/token?grant_type=client_credentials&client_id=?&client_secret=?&scope=?
